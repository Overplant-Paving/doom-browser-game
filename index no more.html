DOOM in Your Browser

Game Controls

  • Arrow keys: Move
  • Ctrl: Fire
  • Alt: Strafe
  • Shift: Run
  • Space: Use/Open
  • 1-9: Select weapons
  • Esc: Menu

Click inside the game area to activate controls. Press F1 for help in game.
